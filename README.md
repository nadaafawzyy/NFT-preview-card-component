# NFT Preview Card Component

A responsive **NFT Preview Card** UI component built from the *Frontend Mentor* challenge.  
This project demonstrates building a clean NFT card layout with image, title, creator, and pricing details using only HTML and CSS.

## 📌 Overview

This repository contains a featured NFT card including:

- NFT artwork image  
- Title and description  
- Creator name  
- Price and time remaining  
- “View on OpenSea” style action button  
- Fully responsive layout
